# Cocoa Framework

Cocoa 是OS X和iOS操作系统的程序运行环境.

NSObject, Foundation & UIKit.

对象生命周期简化 : 
	alloc - init - doSomething - retain - release - release - dealloc

Everything you see on Screen is UIView.
